# Quick Start

Source: https://mjai.app/docs/quickstart

The easiest way to get started is to use the sample code that is already available. Bot tagged users are implemented by Admin and the codes are available on Github.

[Rulebase](https://mjai.app/users/@rulebase) ([source code](https://github.com/smly/mjai.app/blob/main/examples/rulebase/bot.py)) is a simple bot that discards tiles to maximize the ukeire (the number of tiles needed to improve the hand).

- Download: https://github.com/smly/mjai.app/raw/main/examples/rulebase.zip

## Submit your code

You can join the game by submitting your code from [Submission Page](https://mjai.app/submissions).

## Validation

All submissions must at least win against the simplest program that only performs Tsumogiri. Your submission will be automatically validated after it is uploaded.

If the result is not reflected within 5 minutes, there may be a problem. Please feel free to contact [Discussion](https://github.com/smly/mjai.app/discussions).
