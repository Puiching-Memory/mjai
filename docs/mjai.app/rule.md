# Game Rule

Source: https://mjai.app/docs/rule

RiichiLab adopts the Hanchan (半荘; East-South game; 東南戦) of Riichi Mahjong (Japanese Mahjong). The rules are almost same as Tenhou. For the rules of Tenhou, please refer to the [Riichi Wiki](https://riichi.wiki/Tenhou.net_rules).

## Standard Rules

- Kuikae (喰い替え) NOT in use
- Sudden death (サドンデス) brings the game into the next wind. As soon as someone attains 30,000 points (riichi deposits are removed from players' scores), the game ends.
- Sudden death only goes until West Round ends.
- Busting out (飛び) ends the game. Having a negative score ends the game.
- Abortive draws (Nine terminals abortion 九種九牌, four riichi calls 四家立直, three ron calls 三家和了, four kans 四槓散了, four same wind discards 四風連打) allowed.
- Nagashi mangan (流し満貫) allowed.
- Friten riichi (フリテン立直) allowed.
- Pao (パオ) is enforced on daisangen (大三元) and daisuushii (大四喜).

## RiichiLab Unique Rules

- Time limit for thinking is 2 sec
- If a runtime error occurs, the game becomes an abortive draw, and the player pays a mangan penalty to the other participants.
